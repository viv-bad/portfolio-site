About me

I hold a PhD from the Department of Chemistry, University of Cambridge, and two further Master's degrees in Engineering and Physics and Pure Chemistry. Having spent 8 years in a laboratory, specialising in renewable energy technology, I realise the importance of useful, well-designed software in the current world. My first exposure to programming was to write scripts to automate my data processing and visualisation using Python, which was key to my research progression. After finishing my PhD I made it a priority of mine to dive into software engineering head-first and I have uncovered a true passion for building products. I since have completed numerous courses in HTML/CSS, Javascript, NodeJS and ReactJS and have built projects using various technologies which you can find in my skills section.

02

Book Identifier App

As many people do, we own hundreds of books in our apartment. Furthermore, we have many visitors who come to view our bookshelf and want to borrow a book. Therefore, the Book Identifier App was created using JavaScript, NodeJs, Express, MongoDB and Mongoose with an EJS template view engine. This project utilised the MVC architecture along with the OpenLibrary API to identify a given book simply by the ISBN (book-id) number. By entering the ISBN number, an API call is made to the OpenLibrary API where the metadata of the book is retrieved, stored in the database, and displayed on the app. This app has solved the problem of knowing which books we currently have in our apartment, and which we have lent to others.

03

Arquimedes' List

Allocating tasks and chores within a household is always ridden with miscommunication and stress. Arquimedes' List is a task app, where users can create tasks, assign them to a user, and mark them as completed when done. This fullstack app was created using the MongoDB, Express, React and NodeJS (MERN) stack, with the data persisting on a hosted server, communicating with the client through the REST endpoints. This app has made our household much more efficient at delegating and completing tasks.
